Experience
Summer 2019 – Intel Corporation
Intern with Extreme Computing Group within the Intel Data Center Group focused on exascale computing. I delivered three architectural research studies. First, I compared the performance of multiple HPC workloads executing on a high-performance Xeon processor vs. a more area conservative processor with the addition of an accelerator fabric. Second, I scaled down the accelerator fabric to represent an in-core accelerator and compared to the baseline next generation Xeon core. Third I developed components for an asynchronous next generation accelerator fabric using CAST (Caltech Asynchronous Tools). This is a Hardware Defined Language (HDL) with available tools which can synthesize asynchronous logic. Also, developed a wrapper in python to test the functionality of the asynchronous logic and used it to verify that my asynchronous design was fully functional.
Summer 2018: Intel Corporation
Intern with Extreme Computing Group within the Data Center Group focused on exascale computing. Developed a graphical user interface and a Netbatch automation suite of tools for generating and representing performance data on a new data parallel architecture. This tool would take in user input such as what workloads to analyze, what architectural parameters to vary, what graphs to display, etc. and produce graphs that could compare architectural choices in a visually valuable manner. This data was used to optimize the architecture for specific workloads as well as develop a database of information for the general architectural performance information. 2017-2018: IMU Applications
Applied knowledge from Embedded Systems Course to understand and apply an IMU (Inertial Measurement Unit) for use in a self-driving RC model car. Skills developed: best use of available online open source codebases; used open source to accelerate project deliverables while developing best processes for integration and testing; developed UART driver through Arduino to interface with an IMU; developed python based visualizations using Matplotlib to display 3D objects in real time; implemented Quaternions and Euler Angles to implement 3D visualizations.
2016-2017 Piano Player
Developed a robotic two-handed piano player. An engineering project including mechanical, electrical and software components. Key areas included creating original Arduino Music Control Language (AMCL), a compiler which reads in a MIDI file and produces an output file of AMCL, and an Arduino program which reads the AMCL and controls a series of solenoids, which when actuated, depress keys on a piano keyboard. The movement of each “hand” had to be calibrated and optimized based on forward looking of music against different window sizes, and minimizing hand movement. Required researching building different electronic circuits (e.g. flyback diodes and ground noise suppression) and control elements such as pulse width modulation for dynamic sound control. Robotic piano player is fully functional and lots of fun! It won first place in the 2017 Massachusetts State Science Fair at MIT along with over $60K in scholarships.
